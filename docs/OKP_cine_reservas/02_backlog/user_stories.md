# Visitante

* **US-01:** Como visitante, quiero ver la cartelera sin iniciar sesión para conocer las películas disponibles.
* **US-02:** Como visitante, quiero buscar por título y filtrar por género para encontrar una película.
* **US-03:** Como visitante, quiero ver el detalle y las funciones futuras de una película.
* **US-04:** Como visitante, quiero registrarme como cliente.
* **US-05:** Como usuario, quiero iniciar y cerrar sesión.

# Usuario autenticado

* **US-06:** Como usuario autenticado, quiero elegir una función futura.
* **US-07:** Como usuario autenticado, quiero ver el mapa gráfico de la sala.
* **US-08:** Como usuario autenticado, quiero seleccionar uno o varios asientos disponibles.
* **US-09:** Como usuario autenticado, quiero confirmar una reserva sin riesgo de doble asignación.
* **US-10:** Como usuario autenticado, quiero ver mis reservas y su detalle.

# Administrador

* **US-11:** Como administrador, quiero gestionar películas y cargar sus pósteres.
* **US-12:** Como administrador, quiero gestionar salas y conocer su capacidad.
* **US-13:** Como administrador, quiero gestionar funciones sin producir solapamientos.
* **US-14:** Como administrador, quiero recibir mensajes claros cuando una dependencia impida editar o eliminar.
