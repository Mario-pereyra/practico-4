# Fase 1 — Base

Repositorio, configuración, PostgreSQL, migraciones, seed y manejo común de errores.

# Fase 2 — Auth y cartelera

Registro, login, `/auth/me`, logout local, películas públicas y carga de póster administrativa.

# Fase 3 — Salas y funciones

Salas/asientos, CRUD de funciones, `endsAt`, tiempo futuro y solapamiento.

# Fase 4 — Reservas

Mapa de asientos, transacción, restricción única, Mis reservas y detalle.

# Fase 5 — Frontend y QA

Pantallas, roles, estados, responsive, accesibilidad, pruebas de límites y demo completa.

# Fase 6 — Cierre

Sincronizar OpenAPI, documentación, migraciones y trazabilidad; ejecutar el script de coherencia.
