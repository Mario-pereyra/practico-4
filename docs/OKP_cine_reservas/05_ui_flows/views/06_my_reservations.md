# Ruta

`/my-reservations`

# Contenido

* Tarjetas o tabla con película, fecha, sala, asientos y total.
* Orden por fecha de confirmación descendente y enlace “Ver detalle”.
* Empty state para usuario sin reservas.
* Paginación si corresponde.

# Exclusiones

No mostrar pestañas “Canceladas”, botones de cancelación ni reembolsos.
