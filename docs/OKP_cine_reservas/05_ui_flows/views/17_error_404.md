# Error 404

## Objetivo

Mostrar una pantalla clara cuando el recurso o ruta no existe.

## Layout

- `AppShell`.
- Card centrada.
- Mensaje principal.
- CTA hacia cartelera o admin según contexto.

## Mensaje

> No encontramos la página que buscas.

CTA público: “Volver a cartelera”.

CTA admin: “Volver al panel”.

## Reglas UX

- No usar humor excesivo.
- No mostrar stack traces.
- Mantener estética del sistema.

## Prompt Stitch recomendado

Crear una pantalla 404 para una app web de reservas de cine. Debe tener una tarjeta centrada con mensaje “No encontramos la página que buscas” y botón para volver a la cartelera. Estilo sobrio, moderno y claro.

## Checklist de aceptación

- Se muestra en rutas inexistentes.
- Tiene CTA útil.
- No filtra detalles técnicos.
