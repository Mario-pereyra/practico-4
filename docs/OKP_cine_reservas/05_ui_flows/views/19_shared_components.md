# Componentes

* `AppShell` y `AdminShell`.
* `PageHeader`.
* `MovieCard`.
* `DataTable`.
* `FormField`.
* `ConfirmDialog` para eliminación.
* `SeatMap` y `SeatButton`.
* `EmptyState`.
* `ErrorAlert`.
* `LoadingState`.
* `Pagination`.

# Badges

Usar badges para género, clasificación y rol. No se necesita un estado cancelado de reserva.

# Asientos

Estados visuales: disponible, seleccionado y reservado.
