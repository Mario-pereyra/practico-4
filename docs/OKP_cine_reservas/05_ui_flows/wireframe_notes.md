# Cartelera

Tarjetas con póster, título, género, duración, clasificación y acceso al detalle.

# Detalle

Información completa y lista de funciones futuras con fecha, sala y precio.

# Asientos

Grilla con estados disponible, seleccionado y reservado; leyenda, contador y total.

# Mis reservas

Listado sin pestañas de estado ni acciones de cancelación. Cada tarjeta enlaza al detalle.

# Administración

Tablas para películas, salas y funciones; formularios reutilizables; confirmación para eliminar.
