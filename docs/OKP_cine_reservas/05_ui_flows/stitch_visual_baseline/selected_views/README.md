# Uso

Las carpetas contienen `screen.png` y `code.html`. Solo se utilizan para composición visual. Las reglas funcionales provienen de OpenAPI y las vistas Markdown.

# Exclusiones

Las pantallas con limpieza, cancelaciones o reembolsos se movieron a `../legacy_views/`.
