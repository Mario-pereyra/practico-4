# Cartelera

Diseñar una cartelera pública sobria con buscador por título, filtro de género y tarjetas con póster, título, duración, género y clasificación.

# Detalle de película

Mostrar póster, título, sinopsis, duración, género, clasificación y funciones futuras con sala, fecha, hora, precio BOB y acción “Seleccionar asientos”.

# Selección de asientos

Mostrar resumen de función, pantalla, grilla accesible con estados disponible/seleccionado/reservado, leyenda, máximo 20, total BOB y botón “Confirmar reserva”.

# Mis reservas

Mostrar reservas propias con película, fecha, sala, asientos, total y acción “Ver detalle”. No incluir cancelación ni filtros de estados cancelados.

# Administración de funciones

Mostrar listado con película, sala, inicio, fin, precio y acciones. En formulario usar película, sala, inicio y precio; mostrar fin calculado sin margen adicional.

# Restricciones

No agregar pagos, reembolsos, QR, cancelaciones, sucursales ni dashboards complejos.
